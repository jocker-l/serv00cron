# serv00保活服务 php版本

在根目录添加一个`accounts.json`文件，内容如下：

```json
[
  {
    "host": "xxx",
    "port": 22,
    "username": "xxxx",
    "password": "xxxx",
    "command": "",
    "exec_timer": 2
  },
  {
    "host": "xxx",
    "port": 22,
    "username": "xxxx",
    "password": "xxxx",
    "command": "",
    "exec_timer": 2
  }
]
```

command 为执行命令，可以为空，表示不执行命令
exec_timer 为执行命令的间隔时间，单位为秒


-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

1. 安装依赖：
    1. composer install
2. 运行脚本路由：
    1. GET /serv00/start 启动
    2. GET /serv00/stop 停止
